- Tiny


## Usage

1. load module
    ```html
    <!-- Bottom of body -->
    <script type="module" src="https://unpkg.com/@danleyb2/rich-text-editor@0.0.3/dist/es6-unbundled/index.js"></script>
    ```
2. use component
    ```html
    
    <rich-text-editor></rich-text-editor>
    
    ```

3. get value
    ```js
    let v = document.querySelector('rich-text-editor').value;
    console.log(v);
    ```
